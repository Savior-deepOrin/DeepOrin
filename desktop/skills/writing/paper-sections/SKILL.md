# 学术论文写作

指导论文各章节写作、修改和投稿前检查。

## 章节写作指南

### Abstract
- 第一句：问题背景 + 重要性（1句）
- 第二句：现有方法局限性（1句）
- 第三句：你的方法核心思想（1-2句）
- 第四句：关键结果（含数值）
- 第五句：意义/影响（1句）
- **长度**：150-250 words
- **禁忌**：不引用文献、不用缩写、不含未在正文定义的概念

### Introduction
- 第1段：大背景 + 问题重要性
- 第2段：现有方法及其不足（cite 3-5篇）
- 第3段：你的核心贡献（列点1,2,3）
- 第4段：贡献总结 + 论文结构导航
- **禁忌**：不过度吹嘘、不跳过相关工作

### Method
- 结构：Overview → 各组件详解 → 训练/推理细节
- 每个公式：先文字解释再给公式，所有符号必须定义
- 配图：架构图放在 Method 开头
- **公式格式**：$$ 块级，$ 行内

### Experiments
- 主结果表格：加粗最优、下划线次优
- 消融实验：解释每个变化的影响
- 可视化：定性结果至少三个case
- **必须**：实验设置可复现（数据集/超参/硬件）

### Conclusion
- 总结贡献（2-3句）
- 局限性（诚实）
- 未来方向（1-2个具体方向）
- 不加新内容

## 通用写作原则
- Claim-Evidence 对齐：每个声明必须有实验/引用支撑
- 段落结构：1个中心句 + 3-5个支撑句
- 过渡词：However, Therefore, In contrast, Notably
- 时态：相关工作用现在时，你的实验用过去时
- 避免：Obviously, Clearly, Very, Quite

## 投稿前自查清单
- [ ] Abstract 包含具体数值
- [ ] Introduction 明确列出贡献
- [ ] 每个公式符号都有定义
- [ ] 所有表格/图在正文中被引用
- [ ] 实验设置完整可复现
- [ ] 相关工作覆盖近3年
- [ ] 无拼写/语法错误
